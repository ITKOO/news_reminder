* 실행방법
1. config 폴더안에 있는 config.py 파일에서 아래의 내용을 자신의 환경에 맞게 수정합니다.
```  
self.DB_HOST = 'host'
self.DB_USER = 'user_name'
self.DB_PASSWORD = 'password'
self.DB_NAME = 'db_name'
 ```

<br>
 
2. 터미널에서 main.py 파일을 실행합니다.
```  
python main.py
```
